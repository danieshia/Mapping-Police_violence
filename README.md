# Mapping Police Violence
INST327 Group Project
Members: Danieshia Maragh, Sharon Madu, Kimberly Gordon, Rachel Galloway, Ediale Odia 

### Database Description: 
We have designed our database to systematically organize and analyze incidents of fatal police encounters in the United States between 2013 and 2025. The primary goal of this database is to transform complex, unstructured datasets into a structured relational model that supports data-driven insights into patterns of police violence to help support advocacy groups.
